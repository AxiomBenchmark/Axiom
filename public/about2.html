Open Source

In order to allow others to contribute to our project and have others verify our results, we decided to release it as an open source project via GitHub. Our git repository is located on the homepage.

This allows developers to use our tools to add tests for frameworks that they want to see as a part of the project. We have provided a template on our git so that developers can achieve similar results for their devices.
